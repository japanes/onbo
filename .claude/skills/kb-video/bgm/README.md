# Фоновая музыка (BGM)

Опционально. Положи сюда `.mp3`/`.wav` треки, если хочешь подложить музыку под
озвучку. По умолчанию видео собирается **без** музыки (`build ... none`).

Использование:

```bash
python scripts/kb_pipeline.py list-bgm                 # список доступных треков
python scripts/kb_pipeline.py build <outdir> track-01.mp3 --duck 4
```

`--duck N` — насколько тише музыка относительно голоса (в LU, «единицах громкости»;
больше N = тише музыка). Дефолт 4. Трек автоматически зацикливается с кроссфейдом,
если короче видео.

Файлы `*.mp3` в этой папке в `.gitignore` — в репозиторий не попадают.
Права на музыку — на твоей стороне (роялти-фри или собственная).
